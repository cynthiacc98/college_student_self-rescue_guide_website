# 文档目录说明

本目录用于集中管理项目文档，减少根目录杂乱。

## 内容索引

- `OPEN_SOURCE_PRE_FLIGHT_CHECKLIST.md`：开源发布前最终核查清单
- `DESIGN.md`：系统设计说明
- `*_REPORT.md`：功能验证、性能与质量报告
- `*_SUMMARY.md`：阶段性优化总结
- `TODOLIST.md` / `TODO_PERFORMANCE_OPTIMIZATION.md`：任务与优化待办
- `DEBUG_LOG.md`：历史调试记录

## 维护建议

- 新增过程性文档优先放入 `docs/`
- 根目录仅保留入口类文件：`README.md`、`LICENSE`、`Dev-Log.md`、`AGENTS.md`
