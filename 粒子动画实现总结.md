## 🎨 技术实现亮点

### 渲染管线
```
清空画布(拖尾效果) → 
更新粒子物理 → 
计算3D透视 → 
绘制阴影层 → 
绘制发光层(Screen混合) → 
绘制粒子核心 → 
requestAnimationFrame
```

### 物理系统
- **汇聚期**: 螺旋吸引力 + 随机扰动
- **成型期**: 精准定位力
- **交互期**: 鼠标排斥力 + 回弹力
- **开页期**: 3D倾斜变换力

### 3D效果计算
```typescript
// 透视投影
const scale = perspective / (perspective + depth * 1.5)
const rotatedX = x + sin(rotationY) * depth * 0.12
const rotatedY = y + sin(rotationX) * depth * 0.06

// 动态阴影
const shadowOffset = depth * 0.08 + sin(time * 0.002) * 2
```
